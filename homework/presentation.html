Absolute Positioning

Positions

There are four different position values:

static, relative, fixed, Absolute

Elements are then positioned using the top, bottom, left, and right properties.

Position: absolute

Absolutely positioned elements are moved in relation to their closest relatively positioned parent element.

Should a relatively positioned parent element not exist, the absolutely positioned element will be positioned in relation to the body element.

Position: relative

If you want an absolute positioned element to honor the positioning of its parent element the parent element must have position: relative

Because an element is absolutely positioned, it does not sit within the normal flow of the page and will overlap any surrounding elements

Typically, most positioning can be handled without the use of the position properties but in certain cases they can be extremely helpful.
